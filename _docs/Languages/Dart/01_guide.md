# Dart Guide

## Installation

```bash
# Installing
$ brew tap dart-lang/dart
$ brew install dart

# Updating
$ brew update
$ brew upgrade dart
$ brew cleanup dart
```

### Dart Command-line Tools

-   Pub package manager (`pub`)
-   Static analyzer (`dartanalyzer`)
-   Code formater (`dartfmt`)
