## GoLang Guide

```bash
$ export VERSION=1.10  OS=darwin ARCH=amd64
$ wget https://golang.org/dl/go$VERSION.$OS-$ARCH.tar.gz
$ tar -C /usr/local -xzf go$VERSION.$OS-$ARCH.tar.gz
```
